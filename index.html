PHP Media Manager

A downloadable HTML/PHP website for Mac that integrates Sabnzbd, Sonarr, and Radarr APIs with responsive design.

Sonarr Integration

Manage your TV shows with complete Sonarr API integration. View show details, upcoming episodes, and monitor downloads.

Radarr Integration

Browse your movie collection with Radarr API integration. Get detailed information, status updates, and manage downloads.

SABnzbd Integration

Monitor and control your downloads through SABnzbd. View queue status, history, and manage download priorities.

Responsive Design

Enjoy a fully responsive interface that works perfectly on desktop, tablet, and mobile devices.

Easy Configuration

Simple settings page allows you to connect your services with just API keys and URLs. No complex setup required.

Standalone Package

Download and run directly on your Mac without additional dependencies or installations.

Installation Instructions

  1. Download the package using the button above
  2. Unzip the downloaded file
  3. Move the PHP Media Manager folder to your desired location
  4. Open the folder and double-click on "start.command"
  5. Access the interface by navigating to http://localhost:8000 in your browser
  6. Configure your Sonarr, Radarr, and SABnzbd API settings
